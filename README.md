# Sistema de Cadastro de Clientes com Login

Projeto desenvolvido em C# com integração a banco de dados MYSQL, destinado ao gerenciamento de cadastros de clientes. O sistema possui controle de acesso através de login seguro com criptografia de senhas.

## Descrição

O sistema possui duas funcionalidades principais:

1. **Sistema de Login**
   - Validação de usuário e senha.
   - Senhas armazenadas de forma criptografada.
   - Apenas usuários validados podem acessar o sistema de cadastro.

2. **Cadastro de Clientes**
   - CRUD completo: **Criar**, **Visualizar**, **Editar** e **Excluir** clientes.
   - Integração com banco de dados para armazenamento persistente.

## Tecnologias Utilizadas

- **Linguagem:** Visual Studio / C#  - .NET Framework
- **IDE:** Visual Studio  
- **Banco de Dados:**  MySQL
